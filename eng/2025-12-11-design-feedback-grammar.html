English Correction: Design Feedback (Batch 1)

First the image logo doesn't appear.
First, the image logo isn't appearing.

Why?

"Doesn't appear" implies it never happens (general fact). "Isn't appearing" (present continuous) is better for a bug you are observing right now.

I don't want the sentence like "Will Choo Mi-ae 2026 Gyeonggi Province Gubernatorial Election Winner?"
I want to avoid phrasing like "Will Choo Mi-ae...".

Why?

"The sentence like" is slightly unnatural. "Phrasing like" or "A sentence like" is better. "I want to avoid" is a polite but firm way to give negative design feedback.

We need to handle market creation as a batch process on the API server.

You proposed the solution ("batch process") directly. "Handle X as Y" is a strong pattern for technical proposals.
