English Correction: Server Instructions

You should say whether I should rerun api server or not. That goes the same for web server.
Please let me know if I need to restart the API server. The same applies to the web server.

Why?

  • "You should say" → This sounds a bit demanding or directive. "Please let me know" or "Please tell me" is more polite and natural for requests.
  • "rerun" vs "restart" → For servers, we usually say "restart". "Rerun" is typically used for scripts or jobs that finish (like `yarn seed`).
  • "That goes the same for" → This is understandable but slightly informal. "The same applies to" is standard phrasing.

More Natural Variations:

  • "Please specify if a server restart is required for the API or Web." (Concise)
  • "Tell me if I need to restart either the API or the web server." (Direct)
  • "Should I restart the API or web server after these changes?" (Question format)
